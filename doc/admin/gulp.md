###Firing Specific Gulp Tasks

To run all gulp tasks: `gulp`

All other tasks use the 'run' option.

Example: `gulp --run=deployImages`

Available Tasks:  
`runTdd`  
`deployNodeModules`  
`deploySass`  
`deployTemplates`  
`deployImages`  
`deployBaseAngular`  
`deployMt2Js`  
`deployEspApiJs`  
`deployUserJs`  
`deployRoleJs`  
`deployJobJs`  
`deployShowInfoJs`  
`deployWizardJs`  
`deployYmlpCampaignJs`  
`deployClientJs`  
`deployClientGroupJs`   
`deployListProfileJs`
